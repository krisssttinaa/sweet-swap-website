# Sweet Swap

Sweet Swap is a web application that allows users to create, share, and discover a variety of recipes while fostering a community around healthy and dietary-specific cooking.

## Technologies Used

- **Frontend:**
  - React.js
  - HTML5, CSS3
  - Axios for API requests

- **Backend:**
  - Node.js
  - Express.js
  - CockroachDB
  - bcryptjs for password hashing
  - cors for handling Cross-Origin Resource Sharing
  - multer for file uploads
  - (nodemailer for sending emails)
    
## Prerequisites

Ensure you have the following installed on your system:

- Node.js (v16.x.x or later)
- npm (v8.x.x or later)
- MySQL
